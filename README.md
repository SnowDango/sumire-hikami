# Sumire-Hikami -auth0 and reverseproxy server-

## enviroment
- CLIENT_ID: Auth0 client id
- ISSUER_BASE_URL: auth0 issuer base url
- BASE_URL: my app url
- SECRET: auth0 client secret
- SECRET_SESSION: cookie session secret
- PORT: port
